# Luma Weaver Docs

This directory is the long-term home for both user-facing and developer-facing documentation.

The initial structure is audience-based:

- [user/README.md](user/README.md): user-facing docs entry point
- [developer/contributing.md](developer/contributing.md): contributor workflow and expectations
- [developer/architecture.md](developer/architecture.md): workspace and runtime architecture
- [developer/node-authoring.md](developer/node-authoring.md): end-to-end guide for adding and changing nodes

Repository-level companion docs:

- [../README.md](../README.md): project overview, deployment, and development quick start
- [../DOCS.md](../DOCS.md): Home Assistant add-on notes
- [../CHANGELOG.md](../CHANGELOG.md): release history
- [../AGENTS.md](../AGENTS.md): repo-local execution guidance for coding agents

Planned growth areas:

- user documentation for the editor, nodes, and integrations
- reference documentation for protocol types, configuration, and the node catalog
- published docs via GitHub Pages
